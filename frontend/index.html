Pokédex Proxy

Buscador de Pokémon

Esta aplicación consume la PokeAPI a través de un servidor proxy creado con Node.js puro. Escribe el nombre o número de un Pokémon y haz clic en "Buscar".

Información técnica

Backend (proxy) escuchando en:

http://localhost:3000/api/pokemon/{nombre}

El frontend usa Bootstrap 5 y JavaScript ES6, y las peticiones se realizan con fetch hacia el backend, no directamente a PokeAPI.

Ejemplo educativo - Node.js nativo + Bootstrap 5
